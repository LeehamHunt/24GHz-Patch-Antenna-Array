# 24 GHz Microstrip Patch Antenna Array

![Antenna Render](images/antenna_render.png)

## Overview

This project focused on the design and optimization of a 24 GHz microstrip patch antenna array for high-gain microwave applications. The objective was to develop an antenna geometry capable of achieving the desired radiation characteristics while maintaining efficient impedance matching and manufacturability.

The antenna was designed and evaluated using full-wave electromagnetic simulation in CST Studio Suite.

# Design Objectives

- Achieve resonance at 24 GHz
- Maximize realized gain
- Minimize return loss
- Optimize beamwidth
- Improve impedance matching
- Evaluate array spacing effects

---

# Design Methodology

The antenna geometry was developed using standard microstrip patch design equations followed by iterative optimization within CST Studio Suite.

The design process included:

- Initial analytical patch dimension calculations
- Feed network development
- Parametric optimization
- Radiation pattern evaluation
- Impedance matching refinement

# Parametric Analysis

Several antenna parameters were varied to evaluate their effect on performance.

Examples include:

- Patch length
- Patch width
- Feed position
- Array spacing
- Substrate thickness
- Dielectric constant

These studies were used to optimize resonance frequency, gain, and radiation characteristics.
